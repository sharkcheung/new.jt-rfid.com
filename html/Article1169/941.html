RFID超高频（通道门）读写器说明

点击次数:  更新时间:2016-7-28 17:57:33

 

 JT-922通道门是捷通科技开发的新一代超高频（UHF）电子标签读写器。兼容FCC ISM (902～928MHz)标准ETSI(865.6～867.6 MHz)(用户需定制)，用于读写ISO18000-6C（EPC C1 GEN2）的电子标签。本读写器具有体积小巧、性能优越、使用方便、应用广泛、安全可靠、稳定实用等特点。且具有快速防冲突解析和读写处理能力，支持RS232、RS485、RJ45等通讯接口，可直接接驳主机、控制器及其它相关设备，是开放式、大流量、通道式门禁、考勤、签到及过程控制的理想选择。

 产品功能

    JT-922具有读写速度快、读写距离远、可同时操作多个标签、只对专有应用系统的标签进行操作、自动跳频抗干扰能力强等特点。

本读写器具有以下主要功能：

l 读取标签：不仅可以读取标签的ID号，还可读取指定标签存储区的数据；不仅可取单个标签的数据，而且可以同时读取天线波范围内的多个标签的数据。

l 写入标签：可以向指定的标签存储区写入数据。

l 唤醒标签：只有被唤醒的标签才能与读写器进行通信，防止系统外其他标签的干扰，确保读写器与本系统标签信息交换的可靠与准确。

l 锁定标签：对指定的标签存储区设置“锁定”，将使该存储区的数据被锁存并永久“保护”不被改写。并可查询任意存储区(0~31)中的被锁定的存储区地址。

 

JT-922通道门提供两种与PC通信模式(串口和网络) ，可根据用户需要进行切换 。JT-922读写器有4个 射频接口，可外挂四组天线，并可通过软件控制四组天线快速切换（切换时间间隔可设置）。

JT-922提供适合具体应用的API函数，并提供各种工作参数配置接口，用户可通过PC机程序对读写器灵敏度(功率)、工作频率、读取模式等参数进行设置，满足具体环境的使用需求。

 

1.产品规格

性能参数

l   射频功率: 20dBm～30dBm(软件可调)

l   读标签距离:使用10dBi天线＞3m（与标签及环境相关）

l   存储空间：256M大容量Flash掉电保存

l   实时时钟：读写器掉电后，时钟实时保存并走时

l   标签信号值：RSSI数值检测

l   读卡提示：警报器或闪灯

l   触发模式：两路红外触发输入

连接能力

l   电源:110～240V/50～60Hz

l   电源适配器:12V / 3A

l   整机功耗：小于等于16W

l   TCP/IP界面 ：RJ45以太网界面

RFID标准

l   工作频率: 902～928 MHz（中心频率915MHz）其他国家和地区频率标准可定制

l   天线: 圆极化/线性化天线(可根据客户需求设计)

l   支持协议: ISO18000-6C （EPC C1 GEN2）

l   防冲撞: 每秒能读取200张标签

l   适用电子标签: UHF频段EPC global Class 1 Gen2标准标签，如Ti、Philips、UPM、Avery Dennison等Gen2标签

物理参数

l   尺寸: 1500×600×90mm    底座：600x450mm

l   重量: 单个通道门重约25Kg

l   材料: 铝型材

环境参数

l   工作温度: -20~+70 ℃

l   存储温度: -30~+80 ℃

1.2 产品特性

●       盘点装箱货物时，不需要打开箱子即可清点箱内货物；

●       防冲撞强，每秒能读取300张标签；扫描距离远，可达30米(距离可调)；

●       功耗低，发送射频功率(30dBm)时，电流小于1.5A；

●       可直接与嵌入式工控系统连接；

●       提供TCP/IP接口，可直接与办公网络连接，实现一体化网络管理；

●       标准API接口：提供适合具体应用的API 函数，采用标准API接口，支持VC、VB、Delphi等开发环境。

l   海关通关管理：海关通关、转关的物资管理。

l   仓储物流管理：商品流动与仓储管理以及邮件、包裹等的流动管理。

l   门 禁  管 理：包括车辆与人员的进出管理。

l   工艺生产流程：在整个生产过程中监控零部件。

l   设 备  管 理：监控设备的流动情况。
